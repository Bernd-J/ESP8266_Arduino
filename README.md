# BJs Arduino Sketche direkt für den ESP8266
=============================

* folgen demnächst

# worum geht es?

Der SOC (System on Chip) Baustein ESP8266 aus dem Hause Espressif läßt sich seit Anfang 2015 auch direkt mit der Arduino Programmierumgebung bearbeiten.

Erzeugt werden damit sogenannte Sketche welche dann per serieeller Schnittstelle auf den Baustein übertragen werden.

Es gibt unterschiedliche Module am Markt,

von meiner Seite werden momentan eingesetzt:

* ESP8266-7
* ESP8266-12
* ESP8266-12E

Warum?

sie sind Pin- , bzw. Padkompatibel.
